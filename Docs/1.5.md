# 1.5

## 简介

* 修复漏洞 CVE-2018-16843
* 修复漏洞 CVE-2018-16845

### 修复漏洞 CVE-2018-16843

nginx 1.15.6 和 1.14.1 之前的版本的 HTTP/2 实现过程中存在安全漏洞。攻击者可利用该漏 洞消耗大量的内存空间。

**升级Nginx到1.17.2**

### 修复漏洞 CVE-2018-16845

Nginx 1.15.5 及之前的版本和 1.14.1 版本中的 ngx_http_mp4_module 组件存在内存泄露 漏洞，该漏洞源于程序没有正确处理 MP4 文件。远程攻击者可利用该漏洞获取敏感信息或 造成拒绝服务。

**升级Nginx到1.17.2**
